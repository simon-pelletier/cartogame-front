# Electron & React - boilerplate
react - redux - electron - webpack -babel - less - material-ui

## INSTALL
``` bash
$ npm install
```
## DEVELOPMENT
##### Development - Electron
``` bash
$ npm run start
```
##### Development - Web
``` bash
$ npm run web
```
##### Development - Both
``` bash
$ npm run full
```

## DISTRIBUTION
##### Electron
``` bash
$ npm run dist
```
##### Web
``` bash
$ npm run build
```
